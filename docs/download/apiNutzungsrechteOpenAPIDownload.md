# Download OpenAPI Nutzungsrechte

Um es Entwicklern zu vereinfachen die in OpenAPI spezifizierten REST-Schnittstellen in externen Tools zur Automatisierung von Vorgängen zu nutzen, beispielsweise zur automatischen Erzeugung von Client und Server-Stubs, ist es möglich die vollständige Nutzungsrechte-API als eine einzelne YAML Datei im OpenAPI-Format herunterzuladen.

 [OpenAPI Nutzungsrechte als YAML-Datei herunterladen](/dist/openapi/api-nutzungsrechte.yaml)
