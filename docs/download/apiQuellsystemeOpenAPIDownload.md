# Download OpenAPI Quellsysteme

Um es Entwicklern zu vereinfachen die in OpenAPI spezifizierten REST-Schnittstellen in externen Tools zur Automatisierung von Vorgängen zu nutzen, beispielsweise zur automatischen Erzeugung von Client und Server-Stubs, ist es möglich die vollständige Quellsystem-API als eine einzelne YAML Datei im OpenAPI-Format herunterzuladen.

 [OpenAPI Quellsysteme als YAML-Datei herunterladen](/dist/openapi/api-qs.yaml)
