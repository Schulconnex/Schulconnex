# Terminology Report

## docs/datenmodell-qs/anschrift.md

- 'Postleitzahl' — Die Postleitzahl ist ein standardisiertes geografisches Identifikationsmerkmal, das zur Adressierung und Lokalisierung von Orten verwendet wird.
- 'Amtlicher Gemeindename' — Der amtliche Gemeindename ist ein standardisierter Begriff, der den offiziellen Namen einer Gemeinde bezeichnet und in administrativen und geografischen Kontexten verwendet wird.
- 'Ortsteil' — Ein Ortsteil ist eine spezifische Untergliederung eines Ortes und wird in geografischen und administrativen Kontexten verwendet, um eine genauere Lokalisierung zu ermöglichen.

## docs/datenmodell-qs/beziehung.md

- 'Schulbegleiter' — Bezeichnet eine spezifische Rolle im schulischen Kontext, die eine unterstützende Funktion für eine andere Person übernimmt.
- 'Schulbegleiterin' — Bezeichnet eine spezifische Rolle im schulischen Kontext, die eine unterstützende Funktion für eine andere Person übernimmt.
- 'Personenkontext' — Im Sinne der Schnittstelle beschreibt der Personenkontext die spezifische Rolle und Organisation einer Person innerhalb eines bestimmten fachlichen Zusammenhangs.
- 'ktid' — Abkürzung für die ID eines Personenkontexts, die eine eindeutige Zuordnung zu einer Person ermöglicht.
- 'Mandant' — Bezeichnet eine klar abgegrenzte Datenhaltungs- und Verarbeitungseinheit im Schulconnex-Server, die einer Organisation zugeordnet ist.
- 'Schulconnex-Server' — Bezeichnet die zentrale technische Plattform, die die Datenhaltung und Verarbeitung im Schulconnex-System ermöglicht.
- 'Revision' — Im Sinne der Schnittstelle bezeichnet die Revision eine Versionierung eines Datensatzes, die bei Erstellung und Aktualisierung generiert wird.
- 'Codeliste *Beziehungen*' — Bezeichnet eine vordefinierte Liste von Beziehungstypen, die im Schulconnex-Datenmodell verwendet wird.

## docs/datenmodell-qs/datenmodell-arrays.md

- 'JSON' — JSON steht für JavaScript Object Notation und ist ein kompaktes Datenformat für strukturierte Daten. Es wird häufig in der Schnittstellenkommunikation verwendet, insbesondere im Schulconnex-Kontext.
- 'JSON-Array' — Ein JSON-Array ist eine indizierte Liste von Elementen gleichen oder verschiedenen Typs, dargestellt mit eckigen Klammern. Im Schulconnex-Datenmodell werden JSON-Arrays zur Strukturierung von Daten verwendet.
- 'Schulconnex-Server' — Der Schulconnex-Server ist eine zentrale Komponente der Schulconnex-Spezifikation und dient der Bereitstellung und Verarbeitung von Daten im Bildungsbereich.
- 'API' — API steht für Application Programming Interface und bezeichnet eine Schnittstelle, über die Anwendungen miteinander kommunizieren können. Im Schulconnex-Kontext wird die API für den Datenaustausch genutzt.

## docs/datenmodell-qs/datenmodell.md

- 'XBildung' — Ist eine Spezifikation zur Schaffung und Wahrung von Interoperabilität im Bildungswesen in Deutschland, die als Grundlage für das Schulconnex-Datenmodell dient.
- 'XSchule' — Stellt ein schulbezogenes Fachmodul der XBildung-Spezifikation dar und ergänzt diese um schulspezifische Elemente. Es dient als Basis für das Schulconnex-Datenmodell.
- 'Personenkontext' — Bezeichnet den Kontext einer Person, der eine Organisation und eine Rolle umfasst. Im Schulconnex-Datenmodell ist dies ein zentraler Begriff zur Modellierung von Beziehungen.
- 'Sichtfreigabe' — Bezeichnet die Freigabe von Datensätzen für andere Organisationen mit lesendem Zugriff. Im Schulconnex-Datenmodell ist dies eine spezifische Funktion zur Datenfreigabe.
- 'Lernperiode' — Bezeichnet einen Zeitraum, der mit einer Gruppe assoziiert ist. Im Schulconnex-Datenmodell wird dies zur Modellierung von Bildungsabschnitten verwendet.
- 'HTTP' — Das Hypertext Transfer Protocol ist ein Standardprotokoll für die Übertragung von Daten im Internet. Im Schulconnex-Datenmodell wird es für API-Kommunikation verwendet.
- 'String (256)' — Bezeichnet einen Datentyp mit einer maximalen Länge von 256 Zeichen. Im Schulconnex-Datenmodell wird dies zur Definition von Attributen verwendet.
- 'String (32)' — Bezeichnet einen Datentyp mit einer maximalen Länge von 32 Zeichen. Im Schulconnex-Datenmodell wird dies zur Definition von Attributen verwendet.

## docs/datenmodell-qs/datumsformat.md
